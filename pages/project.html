My projects

Cravings App
This app uses the Ionic Framework and React, as well as TypeScript. The purpose of this app is to allow users to create an account and add recipes to their favorites lsit. The recipe page has filters that allows a user to find recipes which use specific ingredients or follow specific diets.
Frequent Flyer
This app uses Java and JSP pages to get query results from a database. The application has a rewards system and allows users to create an account and gain/loose points based on their flight desires. Android Studio is used for creating the app.
Weather Forecast
This program uses C# to make API requests and get the 12 hour forecast. The forecast is then written to a file and used in the future for weather-based applications.
